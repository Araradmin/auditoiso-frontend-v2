Auditoiso - Frontend v2 (React + Vite)

▶ Conectado a backend:
  https://auditoiso-backend-v2.onrender.com/api

▶ Funcionalidades:
  - Registro y login (JWT)
  - Roles (user/admin), sesión persistente
  - Checklist ISO desde el backend
  - Mis auditorías (PDF)
  - Panel Admin: ver usuarios, cambiar roles, ver y eliminar auditorías

▶ Despliegue en Vercel:
  1) https://vercel.com → Add New Project → Upload Project
  2) Sube el contenido de esta carpeta (no el ZIP)
  3) Variables de entorno: no necesarias (API fija en src/services/api.js)
  4) Deploy
